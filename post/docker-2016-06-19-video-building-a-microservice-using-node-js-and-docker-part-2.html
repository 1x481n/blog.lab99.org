构建缓存机制 #

  • 如果存在该层，则用缓存，没有就 rebuild
  • 如果某一层需要 rebuild，其后的所有层都需要 rebuild
  • COPY/ADD 会检查对应文件的校验值，文件内容改变，校验值即发生改变，改变就会 rebuild
  • 其它的命令，看命令的字符串是否有改变，而不去查看命令执行结果是否变化。（这点很重要，比如 apt-get update，其实每次都会不一样，但因为其命令没变化，就不再 rebuild 了）
